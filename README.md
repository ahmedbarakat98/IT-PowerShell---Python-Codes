# IT Automation Scripts Toolkit

A practical collection of **PowerShell** and **Python** scripts designed to help IT professionals, system administrators, helpdesk engineers, and technical support teams automate repetitive tasks, collect system information, troubleshoot common issues, and improve daily workflow efficiency.

---

## Overview

This repository contains useful automation scripts for IT operations, including system checks, reporting, cleanup tasks, file handling, network utilities, and administrative workflows.

The goal of this toolkit is to provide simple, readable, and reusable scripts that can save time and reduce manual work in real IT environments.

---

## What This Repository Includes

- PowerShell scripts for Windows administration
- Python scripts for automation and data processing
- System health checks
- Cache and temporary file cleanup
- Report generation
- Network and device troubleshooting utilities
- Helpdesk and support automation examples
